# OpenCV Camera Project

## 📌 Overview
This project is a simple real-time camera processing program using OpenCV.

It allows switching between different modes such as original, grayscale, and edge detection using keyboard input.

---

## 🎯 Why I built this
- To understand how computer vision works
- To practice real-time video processing
- To build a foundation for AI object detection

---

## ⚙️ Features
- Real-time camera streaming
- Grayscale conversion
- Edge detection (Canny)
- Keyboard control:
  - o: original
  - g: grayscale
  - e: edge detection
  - ESC: exit
